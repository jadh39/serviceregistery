Tchnology Used

  1. Spring Boot
  2. Spring MVC
  3. Spring Data JPA
  4. Gradle
  5. H2 embedded database

How to Run

  1. go to parent folder of the project and run gradle clean build command it will create the jar file "serviceregistry-0.0.1-SNAPSHOT.jar" @ /build/libs folder
  2. goto /build/libs and run java -jar serviceregistry-0.0.1-SNAPSHOT.jar or got to parent directory and run gradle bootRun command

API End Points

1) Create Account

    URI : POST http://localhost:8080/account

    Request Json:

                                {
                                    "AccountId":"xbt123z",
                                     "UserName":"test123"
                                 }

  Response : 201 created                    

   {
    "password": "175bccb7",
    "discription": "Your Account is Opened",
    "success": "true"
    }

2)Register Account


  URI: POST http://localhost:8080/register

  Request Json:

                                {
                                    "url":"http://www.yahoo.co.in",
                                    "AccountId":"abc123z"
                                }

  Response : 202 Accepted                    

  {
    "shortURL": "http://localhost:8080/infobib/c1501d9f"
    }
3)Retrieval of statistics

  URI: GET http://localhost:8080/statistic/{accountId}

   Response : 200 OK                    

{
    "http://www.yahoo.co.in": 1
    }

4) Redirect

enter the short url into the browser it will automatically redirect to Long URL
e.x. enter the url http://localhost:8080/infobib/c1501d9f it will redirect to
http://www.yahoo.co.in
